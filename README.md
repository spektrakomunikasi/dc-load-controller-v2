**by spektrakomunikasi**

### ⚡ Advanced Arduino-Based Electronic Load with Full Protection

---

## ✨ **FITUR UTAMA**

| Fitur | Spesifikasi |
|-------|-----------|
| **Current Range** | 0-50A (6x MOSFET IRF540 Parallel) |
| **Voltage Range** | 0-25V (Protected by Zener 5V1) |
| **Power Rating** | Max 1250W (50A × 25V) |
| **Temperature Control** | Auto Fan PWM 45°C - 90°C |
| **Overvoltage Protection** | Auto-shutdown > 26V |
| **Overtemp Shutdown** | Auto-shutdown > 90°C |
| **Warning Threshold** | Yellow LED blink @ 70°C |
| **Critical Alarm** | Red LED + Buzzer @ 90°C |
| **Display** | LCD 20x4 I2C Real-Time |
| **Measurement Accuracy** | Voltage ±0.1V, Current ±0.2A |
| **Microcontroller** | Arduino Nano (ATmega328P) |
